
# ECEN603

### Project Components
* **DC_ShellScript**: Includes the Synthesis Scripts used to Synthesize all design circuits.
* **Image**: Includes all images used for the project (including ASM Charts).
* **Presentation**: Includes all presentation files.
* **Research Paper**: Includes the final research paper.
* **Testbench**: Includes all testbenches and creation scripts.
* **Verilog SRC**: All Verilog source files used for the design.
* **Report**: Power, area, and timing reports.
